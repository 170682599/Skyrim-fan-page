Skyrim cheats

Here are some cheats that you can use in the game

If you are playing on PC then you can press the ` key to access console commands, here are some commands that you can type in.

Health Cheats

tgm -god mode

tim -immortal mode

player.resethealth -refills health

Leveling up Cheats

psb -Learn every spell

player.advlevel -increases level by 1

player.setlevel # -replace # with the level you want to go to

player.setscale # -replace # with the height you want your character to be

player.setscale # -replace # with the level you want to go to

advskill SkillID # - Replace SkillID with the skill you're modifying and # with the amount you are modifying it by
